# BMI Calculator

A simple React Native application that calculates the Body Mass Index (BMI) based on user inputs (height and weight).

## Features

- Calculate BMI instantly
- Displays weight status (Underweight, Normal, Overweight, Obese)
- Modern UI using gradients

## Requirements

- Node.js >= 14 from the [official website](https://nodejs.org/).
- npm or Expo
- Android/iOS device or simulator

## Installation & Usage

1. Clone or download this repository.
2. Navigate to the project folder.
3. Install dependencies by running: `npm install`
4. Start the development server by running: `npm start`
5. Scan the QR code with the Expo Go app or run on an simulator.
